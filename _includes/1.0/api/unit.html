JSON Format

Units are represented as JSON objects which have the following keys:

{% include table-header.html %}
id {{ site.format.type.integer }} no yes Automatically assigned
name {{ site.format.type.string }} yes yes The Unit Name
type {{ site.format.type.integer }} no yes Describe the kind of unit
1: Piece
2: Weight
3: Dimension
4: Surface
5: Volume
6: Time
