Currencies

JSON Format

Currencies are represented as JSON objects which have the following keys:

{% include table-header.html %}
id {{ site.format.type.integer }} no yes Automatically assigned
iso_code {{ site.format.type.string4 }} yes yes The ISO code of the currency (ISO 4217)
name {{ site.format.type.string64 }} yes yes The Currency Name
main_currency {{ site.format.type.boolean }} no no Is this currency the main accounting for this domain
