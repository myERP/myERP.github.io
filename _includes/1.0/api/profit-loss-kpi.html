Profit loss : kpi

{% highlight bash %}Endpoint: {{ site.api.base_url }}{{ site.api.v1}}{{ site.resource.profit_loss_kpi.url}}{% endhighlight %}

This API is read only.

This endpoint retrieve the overall income, expense and profit / loss for a period.

Options

The following options are available for that endpoint:

{% include table-header-option.html %}
start_date {{ site.format.type.date }} The starting date of the period you want to retrieve.
end_date {{ site.format.type.date }} The ending date of the period you want to retrieve.
show_hint {{ site.format.type.boolean }} Indicate if you want to get a natural language summary based on the given data (Default false).

JSON Format

Main structure

The profit loss kpi is represented as JSON objects which have the following keys:

{% include table-header-readonly.html %}
currency {{ site.format.type.string }} The currency code (i.e. EUR)
data {{ site.format.type.object }} The object that contains the overall KPI.

Overall KPI

Overall KPI are represented as a JSON object which have the following keys:

{% include table-header-readonly.html %}
amount {{ site.format.type.double }} The amount of the same period last year.
progress {{ site.format.type.progress }} Showing the progress between the current and the last year.
