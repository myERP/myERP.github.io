h2. GET taxes/:id

p. Returns infos about a particular tax.

h3. Resource Information

* Rate limited: Yes
* Authentication: Required
* HTTP Methods: GET
* Response Format: JSON

h3. Example Request

{% highlight bash %}
curl -X GET {{ site.api.base_url }}{{ site.api.v1}}{{ site.resource.taxes.url}}/:id {{site.api.auth_pattern}}
{% endhighlight %}

p. JSON response sample

{% highlight bash %}
{
  "id": 5,
  "name": "10% Federal Tax",
  "rate": 10.00
}
{% endhighlight %}

{% highlight bash %}
curl -X GET {{ site.api.base_url }}{{ site.api.v1}}{{ site.resource.taxes.url}} {{site.api.auth_pattern}}
{% endhighlight %}

p. JSON response

The list of all taxes, in a JSON format.
