h1(blue newsection#project). project/:id

h2(#project_get). GET project/:id

p. Returns all the information about the project.

h3. Resource Information

* Rate limited: No
* Authentication: Required
* HTTP Methods: GET
* Response Format: JSON

h3. Example Request

{% highlight bash %}
curl -X GET {{ site.api.base_url }}{{ site.api.v1}}{{ site.resource.project.url}}:id {{site.api.auth_pattern}}
{% endhighlight %}

p. JSON response sample

{% highlight bash %}
{
  "id": 32,
  "name": "Project X",
  "description": "A wonderful project !\nLet's make money."
}
{% endhighlight %}


h2(#project_put). PUT project/:id

p. Update a single project.

h3. Resource Information

* Rate limited: No
* Authentication: Required
* HTTP Methods: PUT
* Body Format: JSON
* Response Format: JSON
* Non editable fields: id

h3. Example Request

{% highlight bash %}
curl -X PUT {{ site.api.base_url }}{{ site.api.v1}}{{ site.resource.project.url}}:id {{site.api.auth_pattern}} -H 'Content-Type: application/json' -T fieldsToUpdate.json
{% endhighlight %}

p. JSON response

The up-to-date JSON of the project you've just modfied.

h1(blue newsection#projects). projects

h2(#projects_get).  GET projects

p. Returns the list of all your projects.

h3. Resource Information

* Rate limited: No
* Authentication: Required
* HTTP Methods: GET
* Body Format: JSON
* Response Format: JSON

h3. Example Request

{% highlight bash %}
curl -X GET {{ site.api.base_url }}{{ site.api.v1}}{{ site.resource.projects.url}} {{site.api.auth_pattern}}
{% endhighlight %}

p. JSON response

The list of all your projects, in a JSON format.


h2(#projects_post). POST projects

p. Create a single project.

h3. Resource Information

* Rate limited: No
* Authentication: Required
* HTTP Methods: POST
* Body Format: JSON
* Response Format: JSON
* Mandatory fields: name

h3. Example Request

{% highlight bash %}
curl -X POST {{ site.api.base_url }}{{ site.api.v1}}{{ site.resource.projects.url}} {{site.api.auth_pattern}} -H 'Content-Type: application/json' -T newProject.json
{% endhighlight %}

p. JSON response

The complete JSON of the project, as it was created by the system.

h3. Resource Information

* Rate limited: No

* HTTP Methods: PUT
* Body Format: JSON
* Response Format: JSON
* Mandatory fields for creation: name
* Non editable fields for edition: id

h3. Example Request

The complete JSON of the project you've just created.


h2(#projects_delete).  DELETE projects

p. Delete several projects in a row

* Rate limited: No
* Authentication: Required
* HTTP Methods: DELETE
* Body Format: JSON
* Response Format: JSON

h3. Example Request

{% highlight bash %}
curl -X DELETE {{ site.api.base_url }}{{ site.api.v1}}{{ site.resource.projects.url}} {{site.api.auth_pattern}} -H 'Content-Type: application/json' -T listOfIDsToDelete.json
