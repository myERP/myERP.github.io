h1(blue newsection#get). Get account

p. Returns all the information about the account contact.

h3. Resource Information

* Rate limited: No
* Authentication: Required
* HTTP Methods: GET
* Response Format: JSON

h3. Example Request

{% highlight bash %}
curl -X GET {{ site.api.base_url }}{{ site.api.v1}}{{ site.resource.account_contact.url}}:id {{site.api.auth_pattern}}
{% endhighlight %}

h1(blue newsection#put). Put account contact

p. Create a new account's contact.

h3. Resource Information

* Rate limited: No
* Authentication: Required
* HTTP Methods: PUT
* Body Format: JSON
* Response Format: JSON
* Mandatory fields: lastname, customer_id

h3. Example Request

{% highlight bash %}
curl -X PUT {{ site.api.base_url }}{{ site.api.v1}}{{ site.resource.account_contacts.url}} {{site.api.auth_pattern}} -H 'Content-Type: application/json' -T jsonFile.json
{% endhighlight %}

p. JSON response

The complete JSON of the account contact you've just created.


h1(blue newsection#delete).  Delete account contact

p. Delete a single account contact.

h3. Resource Information

* Rate limited: No

h3. Example Request

{% highlight bash %}
curl -X DELETE {{ site.api.base_url }}{{ site.api.v1}}{{ site.resource.account_contact.url}}:id {{site.api.auth_pattern}}
{% endhighlight %}

p. JSON response

The JSON of the account contact you've just deleted.
