h1(blue newsection#project). project/:id

h2(#project_get). GET project/:id

p. Returns all the information about the project.

h3. Resource Information

* Rate limited: No
* Authentication: Required
* HTTP Methods: GET
* Response Format: JSON

h3. Example Request

{% highlight bash %}
curl -X GET {{ site.api.base_url }}{{ site.api.v1}}{{ site.resource.project.url}}:id {{site.api.auth_pattern}}
{% endhighlight %}

p. JSON response sample

{% highlight bash %}
{
  "id": 32,
  "name": "Project X",
  "description": "A wonderful project ! Let's make money."
}
{% endhighlight %}


h2(#project_delete). DELETE project/:id

p. Deletes a single project.

h3. Resource Information

* Rate limited: No
* Authentication: Required
* HTTP Methods: DELETE
* Response Format: JSON

h3. Example Request

{% highlight bash %}
curl -X DELETE {{ site.api.base_url }}{{ site.api.v1}}{{ site.resource.project.url}}:id {{site.api.auth_pattern}}
{% endhighlight %}

p. JSON response

The JSON of the project you've just deleted.
