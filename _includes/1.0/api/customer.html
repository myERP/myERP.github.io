Customers

JSON Format

Customers are represented as JSON objects which have the following keys:

{{ site.format.header.field_name }} {{ site.format.header.type }} {{ site.format.header.values }} {{ site.format.header.default }} {{ site.format.header.non_editable }} {{ site.format.header.description }}
id {{ site.format.type.long }} Generated by the system {{ site.format.non_editable }} Code
type {{ site.format.type.long }} 1: Company 2: Individual {{ site.format.mandatory }} Company or Individual
status {{ site.format.type.long }} 1: Customer 2: Lead Customer Lead or Customer
full_name {{ site.format.type.string }} {{ site.format.read_only }} {{ site.format.non_available }} {{ site.format.non_editable }} The full name of this customer
address {{ site.format.type.address }} An empty address Main address
shipping_address {{ site.format.type.address }} An empty address Shipping address
phone_number {{ site.format.type.string64 }} Empty string Phone Number
email {{ site.format.type.string255 }} Empty string Email address
website {{ site.format.type.string255 }} Empty string Website URL
contacts {{ site.format.type.contact }} No Contact {{ site.format.non_editable }} List of this customer's contact
payment_terms_id {{ site.format.type.long }} IDs from Payment terms Due on receipt Payment Terms
currency_iso_code {{ site.format.type.string4 }} ISO Code from your Currencies Your domain main currency Currency
sales_tax_id {{ site.format.type.long }} IDs from Taxes None Sales Tax
user_code {{ site.format.type.string64 }} Anything that does not already belongs to a customer Generated by the system Customer Number
opt_in_to_emails {{ site.format.type.boolean }} true: Yes false: No No Opt-in to Emails
comment {{ site.format.type.text }} Empty String Notes

Company only

If your customer is a company, the JSON object will have the following keys:
{{ site.format.header.field_name }} {{ site.format.header.type }} {{ site.format.header.values }} {{ site.format.header.default }} {{ site.format.header.non_editable }} {{ site.format.header.description }}
company_name {{ site.format.type.string255 }} {{ site.format.mandatory }} Company Name
employee_count {{ site.format.type.integer }} 1 Employee Count
taxpayer_id_number {{ site.format.type.string64 }} Empty String Taxpayer ID Number
siret {{ site.format.type.string64 }} Empty String Siret

Individual only

If your customer is an individual, the JSON object will have the following keys:
{{ site.format.header.field_name }} {{ site.format.header.type }} {{ site.format.header.values }} {{ site.format.header.default }} {{ site.format.header.non_editable }} {{ site.format.header.description }}
first_name {{ site.format.type.string255 }} Empty String First Name
last_name {{ site.format.type.string255 }} {{ site.format.mandatory }} Last Name
middle_name {{ site.format.type.string255 }} Empty String Middle Name
