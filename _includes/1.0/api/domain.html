Domain

{% highlight bash %}Endpoint: {{ site.api.base_url }}{{ site.api.v1}}{{ site.resource.domains.url}}{% endhighlight %}

This endpoint will delete your account and cancel your subscription. The access to that endpoint is restricted to administrators only.

Deleting your domain is a definitive action that cannot be undone. Be advised that all users will no longer be able to use the application nor access datas through the api.

Options

The following options are available for that endpoint:

{% include table-header-option.html %}
reason {{ site.format.type.string }} The reason why the domain is deleted (mandatory).
additionalInfo {{ site.format.type.string }} More information about the reason of the deletion.
