h1(blue newsection#item_family). item_families/:id

h2(#item_family_get). GET item_families/:id

p. Returns the code and the name of the item family.

h3. Resource Information

* Rate limited: No
* Authentication: Required
* HTTP Methods: GET
* Response Format: JSON

h3. Example Request

{% highlight bash %}
curl -X GET {{ site.api.base_url }}{{ site.api.v1}}{{ site.resource.item_families.url}}/:id {{site.api.auth_pattern}}
{% endhighlight %}

p. JSON response sample

{% highlight bash %}
{
  "id": 4,
  "name": "Video Game",
  "code": "FAM-007"
}
{% endhighlight %}

h2(#item_families_put). PUT item_families

p. Create and Update several item families in a row.

h3. Resource Information

* Rate limited: No
* Authentication: Required
* HTTP Methods: PUT
* Body Format: JSON
* Response Format: JSON
* Mandatory fields for creation: name
* Non editable fields for edition: id, code

h3. Example Request

{% highlight bash %}
curl -X PUT {{ site.api.base_url }}{{ site.api.v1}}{{ site.resource.item_families.url}} {{site.api.auth_pattern}} -H 'Content-Type: application/json' -T listOfItemFamilies.json
{% endhighlight %}

A JSON list made of up-to-date info of item families successfully edited or created, and error messages for unsuccessful ones.
