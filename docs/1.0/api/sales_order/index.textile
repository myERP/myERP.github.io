h2. PUT sales_orders/:id

p. Update a single sales order. Use ../order_lines/:id to edit order lines.

h3. Resource Information

* Rate limited: Yes
* Authentication: Required
* HTTP Methods: PUT
* Body Format: JSON
* Response Format: JSON
* Non editable fields: id, user_code, subtotal, reduce_amount, tax_amount, total

h3. Example Request

{% highlight bash %}
curl -X PUT {{ site.api.base_url }}{{ site.api.v1}}{{ site.resource.sales_orders.url}}/:id {{site.api.auth_pattern}} -H 'Content-Type: application/json' -T fieldsToUpdate.json
{% endhighlight %}

p. JSON response

The up-to-date JSON of the sales order you've just modfied.

h2. POST sales_orders

p. Create a single sales order and its order lines.

h3. Resource Information

* Rate limited: Yes
* Authentication: Required
* HTTP Methods: POST
* Body Format: JSON
* Response Format: JSON
* Mandatory fields : customer

h3. Example Request

{% highlight bash %}
curl -X POST {{ site.api.base_url }}{{ site.api.v1}}{{ site.resource.sales_orders.url}} {{site.api.auth_pattern}} -H 'Content-Type: application/json' -T newSalesOrders.json
{% endhighlight %}

p. JSON response

The complete JSON of the sales order, as it was created by the system.



span(anchor#sales_orders_put). ~

h2. PUT sales_orders

p. Create (with order lines) and Update (but not order lines) several sales orders in a row.

h3. Resource Information

* Rate limited: Yes
* Batch limit: {{ site.api.batch_limit }}
* Authentication: Required
* HTTP Methods: PUT
* Body Format: JSON
* Response Format: JSON
* Mandatory fields for creation: customer
* Non editable fields for edition: id, user_code, subtotal, reduce_amount, tax_amount, total

h3. Example Request

{% highlight bash %}
curl -X PUT {{ site.api.base_url }}{{ site.api.v1}}{{ site.resource.sales_orders.url}} {{site.api.auth_pattern}} -H 'Content-Type: application/json' -T listOfSalesOrders.json
{% endhighlight %}

p. JSON response

A JSON list made of up-to-date info of sales orders successfully edited or created, and error messages for unsuccessful ones.
