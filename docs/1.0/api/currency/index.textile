h1(blue). Currency Format

|_. {{ site.format.header.field_name }} |_. {{ site.format.header.description }} |_. {{ site.format.header.type }} |
| id | Code | {{ site.format.type.long }} |

| name | Name | {{ site.format.type.string64 }} |
| main_currency | Is this currency the main accounting for this domain | {{ site.format.type.boolean }} |



span(anchor#currency_get). ~ 

h2. GET currencies/:id

p. Returns all the information about the currency.

h3. Resource Information

* Rate limited: No
* Authentication: Required
* HTTP Methods: GET
* Response Format: JSON

h3. Example Request

{% highlight bash %}
curl -X GET {{ site.api.base_url }}{{ site.api.v1}}{{ site.resource.currencies.url}}/:id {{site.api.auth_pattern}}
{% endhighlight %}

p. JSON response sample

{% highlight bash %}
{
  "id": 133,
  "iso_code": "USD",
  "name": "U.S. Dollar",
  "main_currency": true
}
{% endhighlight %}



span(anchor#currencies_get). ~ 

h2.  GET currencies

p. Returns the list of currencies you have added to your domain.

h3. Resource Information

* Rate limited: No
* Authentication: Required
* HTTP Methods: GET
* Body Format: JSON
* Response Format: JSON

h3. Example Request

{% highlight bash %}
curl -X GET {{ site.api.base_url }}{{ site.api.v1}}{{ site.resource.currencies.url}} {{site.api.auth_pattern}}
{% endhighlight %}

p. JSON response

The list of all the currencies you have added on your domain, in a JSON format.
