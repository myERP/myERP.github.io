Sales: top customers

{% highlight bash %}Endpoint: {{ site.api.base_url }}{{ site.api.v1}}{{ site.resource.sales_top_customers.url }}{% endhighlight %}

This API is read only.

That endpoint allow you to retrieve the list of your top customers during a specific period

Options

The following options are available for that endpoint:

{% include table-header-option.html %}
start_date {{ site.format.type.date }} The starting date of the period you want to retrieve.
end_date {{ site.format.type.date }} The ending date of the period you want to retrieve.
limit {{ site.format.type.integer }} The limit of rows that will be returned (default 5).

JSON Format

Main structure

Sales top customer are represented as JSON objects which have the following keys:

{% include table-header-readonly.html %}
options {{ site.format.type.object }} Options used for the call on this endpoint
period {{ site.format.type.object }} Start and end date used for the call on this endpoint.
rows {{ site.format.type.top_customers }} An array containing the details of top customers.

Top customer

Top customer are represented as JSON objects which have the following keys:

{% include table-header-readonly.html %}
currency {{ site.format.type.string }} The currency code (i.e. EUR).
name {{ site.format.type.string }} The name of the customer.
uri {{ site.format.type.string }} The uri to going to the account of the customer.
value {{ site.format.type.double }} The value invoiced for the period.
previous_value {{ site.format.type.double }} The value invoiced the same period last year.
change_percentage {{ site.format.type.double }} The ratio between the current and the last year.
