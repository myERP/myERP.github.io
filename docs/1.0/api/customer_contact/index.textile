h2. GET customers/:customer_id/contacts/:id

p. Returns all the information about the customer contact. This contact should belongs to the customer _customer_id_.

h3. Resource Information

* Rate limited: Yes
* Authentication: Required
* HTTP Methods: GET
* Response Format: JSON

h3. Example Request

{% highlight bash %}
curl -X GET {{ site.api.base_url }}{{ site.api.v1}}{{ site.resource.customers.url}}/:customer_id{{ site.resource.contacts.url}}/:id {{site.api.auth_pattern}}
{% endhighlight %}

p. Create and Update several customer contacts in a row. All should or will belongs to the customer _customer_id_.

h3. Resource Information

* Rate limited: Yes
* Batch limit: {{ site.api.batch_limit }}
* Authentication: Required
* HTTP Methods: PUT
* Body Format: JSON
* Response Format: JSON
* Mandatory fields for creation: last_name
* Non editable fields for edition: id, customer_id

h3. Example Request

{% highlight bash %}
curl -X PUT {{ site.api.base_url }}{{ site.api.v1}}{{ site.resource.customers.url}}/:customer_id{{ site.resource.contacts.url}} {{site.api.auth_pattern}} -H 'Content-Type: application/json' -T listOfCustomerContacts.json
{% endhighlight %}

p. JSON response

A JSON list made of up-to-date info of contacts successfully edited or created, and error messages for unsuccessful ones.
