Sales: overtime

{% highlight bash %}Endpoint: {{ site.api.base_url }}{{ site.api.v1}}{{ site.resource.sales_overtime.url}}{% endhighlight %}

This API is read only.

This endpoint summaries all your invoices breakdown for a period.

Options

The following options are available for that endpoint:

{% include table-header-option.html %}
start_date {{ site.format.type.date }} The starting date of the period you want to retrieve.
end_date {{ site.format.type.date }} The ending date of the period you want to retrieve.
start_day_of_week {{ site.format.type.integer }} Field number indicating the day of the week.
timezone {{ site.format.type.date }} The timezone (i.e "Europe/Paris").

JSON Format

Main structure

Sales overtime are represented as JSON objects which have the following keys:

{% include table-header-readonly.html %}
periods {{ site.format.type.periods }} The object that containing the cash flow breakdown daily, weekly or monthly.
kpis {{ site.format.type.object }} This object contains the overall counts and totals.
currency {{ site.format.type.string }} The currency code (i.e. EUR).
period_breakdown {{ site.format.type.string }} The period breakdown currently used (i.e. MONTH).

Sales overtime KPI

KPIs are represented as JSON objects which have the following keys:

{% include table-header-readonly.html %}
amount_total {{ site.format.type.double }} The total amount of documents involved in the period.
count_total {{ site.format.type.integer }} The number of documents involved in the period.
data {{ site.format.type.object }} The object containing the amount and the count of each documents involved for the period.
