span(anchor#account_get). ~ 

h2(blue). GET accounts/:id

p. Returns infos about a particular account.

h3. Resource Information

* Rate limited: Yes
* Authentication: Required
* HTTP Methods: GET
* Response Format: JSON

h3. Example Request

{% highlight bash %}
curl -X GET {{ site.api.base_url }}{{ site.api.v1}}{{ site.resource.accounts.url}}/:id {{site.api.auth_pattern}}
{% endhighlight %}

p. JSON response sample

{% highlight bash %}
{
  "id": 100,
  "number": "56100",
  "name": "Freight Charges",
  "is_family": false,
  "family_id": 13
}
{% endhighlight %}


span(anchor#accounts_get). ~ 

h2(blue).  GET accounts

p. Returns all Accounts.

h3. Resource Information

* Rate limited: Yes
* Authentication: Required
* HTTP Methods: GET
* Body Format: JSON
* Response Format: JSON

h3. Example Request

{% highlight bash %}
curl -X GET {{ site.api.base_url }}{{ site.api.v1}}{{ site.resource.accounts.url}} {{site.api.auth_pattern}}
{% endhighlight %}

p. JSON response

The list of all accounts, in a JSON format.
