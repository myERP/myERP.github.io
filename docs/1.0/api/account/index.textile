h1(blue newsection#get). Get account

p. Returns all the information about the account.

h3. Resource Information

* Rate limited: No
* Authentication: Required
* HTTP Methods: GET
* Response Format: JSON

h3. Example Request

{% highlight bash %}
curl -X GET {{ site.api.base_url }}{{ site.api.v1}}{{ site.resource.account.url}}:id {{site.api.auth_pattern}}

h1(blue newsection#put). Put account

p. Create a new account.

h3. Resource Information

* Rate limited: No
* Authentication: Required
* HTTP Methods: PUT
* Body Format: JSON
* Response Format: JSON
* Mandatory fields: type, status, company_name (for Company)/lastname (for Individual)

h3. Example Request

{% highlight bash %}
curl -X PUT {{ site.api.base_url }}{{ site.api.v1}}{{ site.resource.accounts.url}} {{site.api.auth_pattern}} -H 'Content-Type: application/json' -T jsonFile.json
{% endhighlight %}

p. JSON response

The complete JSON of the account you've just created.


h1(blue newsection#delete).  Delete account

h3. Resource Information

* Rate limited: No

h3. Example Request

{% endhighlight %}

p. JSON response
