{% highlight bash %}Endpoint: {{ site.api.base_url }}{{ site.api.v1}}{{ site.resource.installments.url}}{% endhighlight %}

This API is read only.

JSON Format

Installments are represented as JSON objects which have the following keys:

{% include table-header-readonly.html %}
id {{ site.format.type.integer }} Automatically assigned.
invoice_id {{ site.format.type.integer }} Id of the related Invoice.
due_date {{ site.format.type.date }} The expected payment date.
Format: {{ site.format.date }}
amount {{ site.format.type.double }} The installment initial amount.
outstanding_amount {{ site.format.type.double }} Amount left to settle.
An outstanding amount of 0.00 means that the Installment has been fully paid.
currency_iso_code {{ site.format.type.string }} The ISO code of the currency (ISO 4217).
payment_request_id {{ site.format.type.integer }} Id of the related PaymentRequest used with integrated payment solutions.
