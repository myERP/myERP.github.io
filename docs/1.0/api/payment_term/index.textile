span(anchor#payment_term_get). ~ 

h2. GET payment_terms/:id

p. Returns the code and the name of the payment term.

h3. Resource Information

* Rate limited: No
* Authentication: Required
* HTTP Methods: GET
* Response Format: JSON

h3. Example Request

{% highlight bash %}
curl -X GET {{ site.api.base_url }}{{ site.api.v1}}{{ site.resource.payment_terms.url}}/:id {{site.api.auth_pattern}}
{% endhighlight %}

p. JSON response sample

{% highlight bash %}
{
  "id": 4,
  "name": "In two installments"
}
{% endhighlight %}

h2.  GET payment_terms

p. Returns the list of all your payment terms.

h3. Resource Information

* Rate limited: No
* Authentication: Required
* HTTP Methods: GET
* Body Format: JSON
* Response Format: JSON

h3. Example Request

{% highlight bash %}
curl -X GET {{ site.api.base_url }}{{ site.api.v1}}{{ site.resource.payment_terms.url}} {{site.api.auth_pattern}}
{% endhighlight %}

p. JSON response

The list of all your payment terms, in a JSON format.
