h2(#accounting_transaction_get). GET accounting_transactions/:id

p. Returns all the information about the accounting transaction.

h3. Resource Information

* Rate limited: No
* Authentication: Required
* HTTP Methods: GET
* Response Format: JSON

h3. Example Request

{% highlight bash %}

{% endhighlight %}

h2(#accounting_transaction_delete). DELETE accounting_transactions/:id

p. Deletes a single accounting transaction. Only manual accounting transactions can be deleted. In order to delete other transactions, you need to deleted the related document (Invoice, Payment, etc...).

h3. Resource Information

* Rate limited: No
* Authentication: Required
* HTTP Methods: DELETE
* Response Format: JSON

h3. Example Request

{% highlight bash %}
curl -X DELETE {{ site.api.base_url }}{{ site.api.v1}}{{ site.resource.accounting_transactions.url}}/:id {{site.api.auth_pattern}}
{% endhighlight %}

p. JSON response

The JSON of the accounting transaction you've just deleted.


h1(blue newsection#accounting_transactions). accounting_transactions

h2(#accounting_transactions_get).  GET accounting_transactions

p. Returns the list of all your accounting transactions.

h3. Resource Information

* Rate limited: No
* Authentication: Required
* HTTP Methods: GET
* Body Format: JSON
* Response Format: JSON

h3. Example Request

{% highlight bash %}
curl -X GET {{ site.api.base_url }}{{ site.api.v1}}{{ site.resource.accounting_transactions.url}} {{site.api.auth_pattern}}
{% endhighlight %}

p. JSON response

The list of all your accounting transactions, in a JSON format.


h2(#accounting_transactions_post). POST accounting_transactions

p. Create a single accounting transaction.

h3. Resource Information

* Rate limited: No
* Authentication: Required
* HTTP Methods: POST
* Body Format: JSON
* Response Format: JSON
* Mandatory fields : piece_number, entries

h3. Example Request

{% highlight bash %}
curl -X POST {{ site.api.base_url }}{{ site.api.v1}}{{ site.resource.accounting_transactions.url}} {{site.api.auth_pattern}} -H 'Content-Type: application/json' -T newAccountingTransaction.json
{% endhighlight %}

p. JSON response

The complete JSON of the accounting transaction, as it was created by the system.


h2(#accounting_transactions_delete).  DELETE accounting_transactions

p. Delete several accounting transactions in a row. Only manual accounting transactions can be deleted. In order to delete other transactions, you need to deleted the related document (Invoice, Payment, etc...).

* Rate limited: No
* Authentication: Required
* HTTP Methods: DELETE
* Body Format: JSON
* Response Format: JSON

h3. Example Request

{% highlight bash %}
curl -X DELETE {{ site.api.base_url }}{{ site.api.v1}}{{ site.resource.accounting_transactions.url}} {{site.api.auth_pattern}} -H 'Content-Type: application/json' -T listOfIDsToDelete.json
{% endhighlight %}

p. JSON response

A JSON list made of the complete info of accounting_transactions successfully deleted, and error messages for unsuccessful ones.
