span(anchor#project_get). ~

h2. GET projects/:id

p. Returns all the information about the project.

h3. Resource Information

* Rate limited: Yes
* Authentication: Required
* HTTP Methods: GET
* Response Format: JSON

h3. Example Request

{% highlight bash %}
curl -X GET {{ site.api.base_url }}{{ site.api.v1}}{{ site.resource.projects.url}}/:id {{site.api.auth_pattern}}
{% endhighlight %}

p. JSON response sample

{% highlight bash %}
{
  "id": 32,
  "name": "Project X",
  "description": "A wonderful project !\nLet's make money."
}
{% endhighlight %}

h2. PUT projects/:id

p. Update a single project.

h3. Resource Information

* Rate limited: Yes
* Authentication: Required
* HTTP Methods: PUT
* Body Format: JSON
* Response Format: JSON
* Non editable fields: id

h3. Example Request

{% highlight bash %}
curl -X PUT {{ site.api.base_url }}{{ site.api.v1}}{{ site.resource.projects.url}}/:id {{site.api.auth_pattern}} -H 'Content-Type: application/json' -T fieldsToUpdate.json
{% endhighlight %}

p. JSON response

The up-to-date JSON of the project you've just modfied.
