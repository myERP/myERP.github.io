Sales: top invoiced items

{% highlight bash %}Endpoint: {{ site.api.base_url }}{{ site.api.v1}}{{ site.resource.sales_top_invoiced.url}}{% endhighlight %}

This API is read only.

That endpoint allow you to retrieve the list of your top invoiced items during a specific period

Options

The following options are available for that endpoint:

{% include table-header-option.html %}
start_date {{ site.format.type.date }} The starting date of the period you want to retrieve.
end_date {{ site.format.type.date }} The ending date of the period you want to retrieve.
limit {{ site.format.type.integer }} The limit of rows that will be returned (default 5).

JSON Format

Main structure

Sales overtime are represented as JSON objects which have the following keys:

{% include table-header-readonly.html %}
options {{ site.format.type.object }} Options used for the call on this endpoint
period {{ site.format.type.object }} Start and end date used for the call on this endpoint.
rows {{ site.format.type.top_invoiced }} An array containing the details of top invoiced items.

Top customer

Top customer are represented as JSON objects which have the following keys:

{% include table-header-readonly.html %}
amount {{ site.format.type.double }} The amount invoiced for the period.
amount_currency {{ site.format.type.string }} The currency code (i.e. EUR).
name {{ site.format.type.string }} The name of the product / service.
uri {{ site.format.type.string }} The uri to going to the product / service.
value {{ site.format.type.double }} The number of product / service invoiced for the period.
previous_value {{ site.format.type.double }} The value of the same period last year.
change_percentage {{ site.format.type.double }} The ratio between the current and the last year.
