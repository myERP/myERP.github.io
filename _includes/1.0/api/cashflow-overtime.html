Cash flow: overtime

{% highlight bash %}Endpoint: {{ site.api.base_url }}{{ site.api.v1 }}{{ site.resource.cash_flow_overtime.url }}{% endhighlight %}

This API is read only.

This endpoint provide you the cash flow breakdown daily, weekly or monthly for a period.

Options

The following options are available for that endpoint:

{% include table-header-option.html %}
start_date {{ site.format.type.date }} The starting date of the period you want to retrieve.
end_date {{ site.format.type.date }} The ending date of the period you want to retrieve.
period_breakdown {{ site.format.type.string }} How the cash flow will be breakdown. Value among : DAY, WEEK and MONTH (default)
timezone {{ site.format.type.date }} The timezone (i.e "Europe/Paris").

JSON Format

Main structure

The cash flow overtime is represented as JSON objects which have the following keys:

{% include table-header-readonly.html %}
periods {{ site.format.type.periods }} The object that containing the cash flow breakdown daily, weekly or monthly.
kpis {{ site.format.type.object }} This object contains the overall counts and totals.
currency {{ site.format.type.string }} The currency code (i.e. EUR).
period_breakdown {{ site.format.type.string }} The period breakdown currently used (i.e. MONTH).

Periods

Periods are represented as JSON objects which have the following keys:

{% include table-header-readonly.html %}
amount_total {{ site.format.type.integer }} The amount total for this period.
count_total {{ site.format.type.integer }} The number of documents involved in this period.
data {{ site.format.type.object }} This object contains the amount and the count of each document for this period.
