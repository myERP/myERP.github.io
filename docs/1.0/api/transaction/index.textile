h2. GET transactions/:id

p. Returns all the information about the transaction.

h3. Resource Information

* Rate limited: Yes
* Authentication: Required
* HTTP Methods: GET
* Response Format: JSON

h3. Example Request

{% highlight bash %}
curl -X GET {{ site.api.base_url }}{{ site.api.v1}}{{ site.resource.transactions.url}}/:id {{site.api.auth_pattern}}
{% endhighlight %}

span(anchor#transaction_delete). ~

h2. DELETE transactions/:id

p. Deletes a single transaction. Only manual transactions can be deleted. In order to delete other transactions, you need to deleted the related document (Invoice, Payment, etc...).

h3. Resource Information

* Rate limited: Yes
* Authentication: Required
* HTTP Methods: DELETE
* Response Format: JSON

h3. Example Request

{% highlight bash %}
curl -X DELETE {{ site.api.base_url }}{{ site.api.v1}}{{ site.resource.transactions.url}}/:id {{site.api.auth_pattern}}
{% endhighlight %}

p. JSON response

The JSON of the transaction you've just deleted.


span(anchor#transactions_get). ~

h2.  GET transactions

p. Returns the list of all your transactions.

h3. Resource Information

* Rate limited: Yes
* Authentication: Required
* HTTP Methods: GET
* Body Format: JSON
* Response Format: JSON

h3. Example Request

{% highlight bash %}
curl -X GET {{ site.api.base_url }}{{ site.api.v1}}{{ site.resource.transactions.url}} {{site.api.auth_pattern}}
{% endhighlight %}

p. JSON response

The list of all your transactions, in a JSON format.



span(anchor#transactions_post). ~

h2. POST transactions

p. Create a single transaction.

h3. Resource Information

* Rate limited: Yes
* Authentication: Required
* HTTP Methods: POST
* Body Format: JSON
* Response Format: JSON
* Mandatory fields : piece_number, entries

h3. Example Request

{% highlight bash %}
curl -X POST {{ site.api.base_url }}{{ site.api.v1}}{{ site.resource.transactions.url}} {{site.api.auth_pattern}} -H 'Content-Type: application/json' -T newTransaction.json
{% endhighlight %}

p. JSON response

The complete JSON of the transaction, as it was created by the system.


span(anchor#transactions_delete). ~


h2.  DELETE transactions

p. Delete several transactions in a row. Only manual transactions can be deleted. In order to delete other transactions, you need to deleted the related document (Invoice, Payment, etc...).

* Rate limited: Yes
* Batch limit: {{ site.api.batch_limit }}
* Authentication: Required
* HTTP Methods: DELETE
* Body Format: JSON
* Response Format: JSON

h3. Example Request

{% highlight bash %}
curl -X DELETE {{ site.api.base_url }}{{ site.api.v1}}{{ site.resource.transactions.url}} {{site.api.auth_pattern}} -H 'Content-Type: application/json' -T listOfIDsToDelete.json
{% endhighlight %}

p. JSON response

A JSON list made of the complete info of transactions successfully deleted, and error messages for unsuccessful ones.
