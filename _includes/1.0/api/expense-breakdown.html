Expense : breakdown

{% highlight bash %}Endpoint: {{ site.api.base_url }}{{ site.api.v1}}{{ site.resource.expense_breakdown.url}}{% endhighlight %}

This API is read only.

This endpoint list your expenses for the chosen period.

Options

The following options are available for that endpoint:

{% include table-header-option.html %}
start_date {{ site.format.type.date }} The starting date of the period you want to retrieve.
end_date {{ site.format.type.date }} The ending date of the period you want to retrieve.

JSON Format

Main structure

The expense breakdown is represented as JSON objects which have the following keys:

{% include table-header-readonly.html %}
progress {{ site.format.type.progress }} Showing the progress between the current and the last year.
currency {{ site.format.type.string }} The currency code (i.e. EUR)
values {{ site.format.type.expenses }} The list of the expenses for the period.
amount_total {{ site.format.type.double }} The total amount expensed for the period.

Expense

Expense are represented as a JSON object which have the following keys:

{% include table-header-readonly.html %}
id {{ site.format.type.integer }} Automatically assigned.
amount {{ site.format.type.double }} The amount of the expense.
label {{ site.format.type.string }} The description of the expense.
progress {{ site.format.type.progress }} Showing the progress between the current and the last year.

Progress

Progress are represented as a JSON object which have the following keys:

{% include table-header-readonly.html %}
previous_amount {{ site.format.type.double }} The amount of the same period last year.
change_percentage {{ site.format.type.double }} The ratio between the current and the last year.
